# MUREO Legal Pages

Public support, privacy policy, and terms pages for the iOS app 幕里（MUREO）.

## GitHub Pages URLs

After GitHub Pages is enabled with source `main` / `docs`, the public URLs should be:

- Home: https://vicchoo.github.io/mureo-legal/
- Support URL: https://vicchoo.github.io/mureo-legal/support.html
- Privacy Policy URL: https://vicchoo.github.io/mureo-legal/privacy.html
- Terms: https://vicchoo.github.io/mureo-legal/terms.html

## App Store Connect

Use these fields:

- Support URL: `https://vicchoo.github.io/mureo-legal/support.html`
- Privacy Policy URL: `https://vicchoo.github.io/mureo-legal/privacy.html`
- Marketing URL: `https://vicchoo.github.io/mureo-legal/`

Review note suggestion:

```text
This app is local-first and does not require account login.
Core museum content is stored on-device.
There is no cloud sync in the current public version.
Support URL: https://vicchoo.github.io/mureo-legal/support.html
Privacy Policy URL: https://vicchoo.github.io/mureo-legal/privacy.html
Terms: https://vicchoo.github.io/mureo-legal/terms.html
```
